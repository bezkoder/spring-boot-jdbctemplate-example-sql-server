# Spring Boot JdbcTemplate example with SQL Server: Build CRUD Rest API
Build a Spring Boot CRUD Rest API example that uses Spring Data Jdbc to make CRUD Operations with SQL Server Database. You'll know:
- How to configure Spring Data to work with SQL Server database
- How to define Data Models and Repository interfaces
- Way to create Spring Rest Controller to process HTTP requests
- Way to use JdbcTemplate to interact with Database
